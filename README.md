# LegacyDock

LegacyDock is an open-source, local-first operating center for legacy Apple devices running iOS 6 through iOS 9.

It is not intended to be another jailbreak tool or package manager clone. The goal is to provide a polished management platform for preserving, maintaining, repairing, enhancing, and documenting legacy Apple devices.

## Prototype

This repository currently contains a static product prototype:

- device dashboard
- device-aware compatibility scores
- package marketplace with safety review
- repository health tools
- local snapshot timeline
- preservation archive manifest
- optional cloud workspace positioning

Open `index.html` locally to view the prototype, or enable GitHub Pages for the repository root.

## Product Direction

The longer-form product and technical plan lives in [docs/legacydock-blueprint.md](docs/legacydock-blueprint.md).

## Principles

- Open source local core
- Local-first and useful without an account
- Privacy focused
- Safety before convenience
- Transparent compatibility and risk explanations
- Optional cloud services for sync, backup, and community intelligence
- Long-term preservation of legacy hardware

## Status

Early product prototype. The current site is static HTML, CSS, and JavaScript intended to communicate the desired product experience before native desktop and device integration work begins.
