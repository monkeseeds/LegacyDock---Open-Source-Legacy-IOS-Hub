Overview

LegacyDock Desktop

LegacyDock is designed as a local-first operating center for iOS 6 through iOS 9 devices. The web UI you see now is the shared product surface for the future desktop shell.

Quickstart

Try the current build

  1. Open the landing page and start in dark theme by default.
  2. Launch the console to inspect demo devices and iOS compatibility coverage.
  3. Review a package to see firmware, device, dependency, conflict, and community checks.
  4. Capture a snapshot or export a preservation report from the console.

Core concepts

The product model

Local core

Device detection, repository parsing, snapshots, reports, and safety checks should work without an account.

Compatibility catalog

LegacyDock currently indexes every iOS 6.0 through 9.3.6 release in the demo compatibility engine.

Restoration Lab

Advanced jailbreak, restore, SHSH, and ramdisk flows are handled as credited, explicit external handoffs.

Safety model

Snapshot first, explain risk, require consent

LegacyDock should not silently mutate a device. A real production action should show exact firmware, model, dependencies, conflicts, recovery options, and a snapshot recommendation before execution.

Schemas

Stable records for preservation

The repository includes JSON schemas for device profiles, packages, repositories, snapshots, restoration workflows, and community reports.

Integrations

Respect upstream work

LegacyDock credits Legacy iOS Kit and iOS Obscura Server as relevant prior art. The current repository does not vendor their code.
