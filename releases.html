LegacyDock Desktop Preview

Tag v0.1.0 · rebuilt Windows installers · updater feed wired

Pre-release

What changed

  • Rebuilt the current Windows installer set and removed stale smoke-test installers so the release folder only keeps the fresh 0.1.0 NSIS and MSI artifacts.
  • Wired the desktop updater to the hosted Supabase manifest at legacydock-updates/latest.json and verified the in-app update check flow.
  • Fixed desktop clipping issues across the reports screen, updater panel, privacy controls, and Device Doctor footnotes.
  • Kept the full desktop setup wizard in place: device detection, compatibility scan, repository health, setup checklist, Device Doctor, service restoration, and final summary export.
  • Preserved the legacy Apple/Cydia-inspired brushed-metal UI pass across the site and the Tauri app shell.
  • Kept the pricing cleanup from removing the Studio card, folding yearly billing into LegacyDock Care, and setting the yearly price at $49.99.
  • Kept all device-changing actions manual and guarded; the current preview is local-first and diagnostic-first.

Release status

  • Windows packaging: working locally with NSIS and MSI outputs.
  • Updater feed: connected to the hosted Supabase public manifest.
  • Updater signing: active for update artifacts.
  • Windows code signing: still pending, so SmartScreen warnings are still expected.
  • macOS installer: planned next after Windows hardening.

Install notes

These preview installers are updater-signed but not yet Windows code-signed. Windows SmartScreen may still warn until the production certificate is added.

Next release blockers

  • Windows code-signing certificate wiring for trusted installer delivery.
  • Real-device QA with libimobiledevice against iOS 6 through iOS 9 hardware.
  • Supabase auth, storage structure, and row-level security for cloud-backed features.
  • Final privacy, terms, export/delete, and telemetry consent review.
  • Third-party license review before bundling external tools or mirrored assets.
  • Beta release checklist covering install, update, rollback, and consent regressions.
