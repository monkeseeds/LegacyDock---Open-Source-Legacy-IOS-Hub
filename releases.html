LegacyDock Desktop Preview

Tag v0.1.0 · Tauri desktop scaffold · setup wizard preview

Pre-release

What changed

  • Added Tauri desktop app scaffold with React, Vite, Tailwind, and Rust command modules.
  • Added Desktop Setup Wizard for detection, compatibility, repository health, setup checklist, Device Doctor, service restoration, and final report export.
  • Configured Windows and macOS artifact jobs through GitHub Actions.
  • Kept device-changing actions disabled; current setup guidance is manual and local-first.

Install notes

Early artifacts are unsigned until signing certificates and Apple notarization credentials are configured. Windows SmartScreen and macOS Gatekeeper may warn on preview builds.
