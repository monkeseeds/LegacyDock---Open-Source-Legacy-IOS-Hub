Desktop Releases

Windows first. macOS next. Linux later.

LegacyDock is being packaged as a Tauri desktop app with a Rust backend and React setup wizard. Release artifacts are built through GitHub Actions so Windows and macOS installers can be produced on their native runners.

Build installers Release notes
Windows

.exe

Primary desktop target. Produces NSIS setup `.exe` and MSI artifacts from the Windows release workflow.

Build Windows Installer
  • Target: Windows 10/11 x64
  • Bundle: NSIS `.exe`
  • Bundle: MSI
  • Status: unsigned until certificate setup
macOS

.dmg

Second desktop target. Produces a macOS DMG from the macOS release workflow runner.

Build macOS DMG
  • Target: Apple Silicon and Intel runners
  • Bundle: DMG
  • Status: unsigned until Apple Developer setup
  • Notarization required before stable release
Linux

Later

Linux packaging is intentionally deferred until Windows and macOS release flows are stable.

View Roadmap
  • Future: AppImage
  • Future: Deb
  • Requires Linux QA
  • Not part of first release pass
Updater

Feed

Auto-update feed scaffolding exists, but stable updates stay disabled until artifacts are signed.

View Feed
  • Feed: `updates/stable.json`
  • Requires Tauri signing key
  • Requires signed installers
  • Dev artifacts are manual downloads
