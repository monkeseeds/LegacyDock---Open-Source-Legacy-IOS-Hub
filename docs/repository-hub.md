# Legacy Repository Hub And Community Resources

LegacyDock now includes a Repository Hub inside the console. Its purpose is to reduce scattered searching across Reddit, Discord, old forums, and stale package lists.

## Included Surfaces

- Searchable repository browser.
- Repository health status, last check, response notes, SSL/HTTP support, package count, category, maintainer, rating, and verification state.
- Add to Cydia links and copy URL actions.
- Browser-safe manual health refresh using the local community database.
- Package browser with version, dependency, compatibility, repository, rating, verification, and last-updated metadata.
- Essential tweaks database for utilities, recording, UI, restoration, and applications.
- Legacy service catalog for TubeRepair, MapsX, Weather, iTunes Store, App Store, push notifications, Siri, Game Center, and YouTube restoration research.
- Repository submission form staged for future moderation.
- Dead repository archive with original URLs, mirror status, and recovery instructions.
- Community resource center for r/LegacyJailbreak, iPhoneOS Obscura, Legacy iOS Kit, and related trusted resources.
- Device-specific smart recommendations based on selected model and firmware.

## Device Doctor Integration

Device Doctor uses the repository catalog to detect stale sources, duplicate mirrors, missing recommended repositories, broken package metadata, and compatible restoration packages.

Current GitHub Pages behavior is local and fixture-backed. A future desktop or hosted service can replace these local health values with live repository probes, moderation queues, and signed community reports.
