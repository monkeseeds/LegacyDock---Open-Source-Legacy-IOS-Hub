Why Desktop

The console is local-first by design.

A browser page can show the product, but the real console needs native permissions, local storage, USB detection, and guarded repair workflows. That is why the web page is a showcase and the Tauri app is the working surface.

Read-only inspectionUSB detection and package-state parsing before any install planning.
