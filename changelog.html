Device Doctor

Device Doctor and LegacyDock Care intelligence

Device Doctor now has explainable health scores, richer diagnostics, and a LegacyDock Care layer for repair intelligence.

  • Added performance, stability, battery, storage, compatibility, repository, and overall health scores.
  • Added Care repair plans, community intelligence, bootloop risk, smart alternatives, modernization guidance, snapshot intelligence, and health timeline.
  • Updated pricing around the model that scanning is free and intelligence is premium.

June 15, 2026
Local core

Commercial readiness foundation

LegacyDock now has executable local-core modules for device discovery, repository parsing, dependency resolution, safety-gated install planning, and workspace persistence.

  • Added a `libimobiledevice` adapter boundary for USB device discovery.
  • Added Debian `Packages` index parsing and package relationship normalization.
  • Added dependency planning and explicit install-operation safety gates.
  • Added CLI commands and automated tests for the core flows.
